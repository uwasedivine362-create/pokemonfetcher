
The Pokémon Fetcher Web Application is a responsive website that allows users to search for Pokémon by name or ID and view their information. The application gets data from the PokéAPI and shows details like Pokémon images, types, height, weight, and experience.

The project is built using HTML for structure, JavaScript for fetching and displaying data, and Tailwind CSS for styling and responsive design.

Main Features

Search Pokémon by name or ID

Display Pokémon images, types, and stats

Responsive design for mobile and desktop

Loading spinner during API requests

Error messages for invalid searches

Animations and hover effects

What the Project Teaches

The project helps developers learn:

How to fetch data from an API

How to use async/await for asynchronous programming

How to manipulate the DOM dynamically

How to work with JSON data

How to design interfaces using Tailwind CSS

How to handle errors in web applications

How to Run the Project

The application can be run by:

Opening the index.html file directly

Using Live Server in VS Code

Running a local Python server

Folder Structure

The project contains:

index.html – the main webpage

script.js – JavaScript logic and API calls

README.md – project documentation
